# Minimum

- Linux server
- 1 Core
- 1GB RAM
- 512MB SWAP
- 5GB for system
- 5GB for Seafile data 


# Recommended

- Linux server
- 2 Cores
- 2GB or more
- 1GB SWAP or more
- 15GB+ for system
- 100GB+ for Seafile data

It is recommended that you use multiple disks in a SW/HW RAID to provide better I/O performance.
